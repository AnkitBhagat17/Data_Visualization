## 🚀 Example Usage
```python
import matplotlib.pyplot as plt

# Example: Linear Plot
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]
plt.plot(x, y)
plt.title("Linear Plot Example")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.show()
